# Smart Calculator (C++)

A command-line based smart calculator built using C++.  
This project demonstrates core programming concepts like functions, exception handling, and data structures.

---

## Features

- Basic operations (+, -, *, /)
- Error handling (division by zero, invalid operator)
- Continuous calculations
- History tracking of previous calculations
- Clean formatted output

---

## Technologies Used

- C++
- STL (vector, string)
- Exception Handling
- String Streams (`ostringstream`)

---

## Future Improvements

- Expression parsing (2 + 3 * 4)
- Support for parentheses
- GUI version
- Scientific calculator functions
